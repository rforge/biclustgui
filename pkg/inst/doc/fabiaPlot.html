RcmdrPlugin.BiclustGUI

Graphical Displays for FABIA

Several plots available from the fabia package.



FABIA Plots:

  1. Biplot of a Matrix Factorization result.
       

  1. Information content of biclusters.
  2. Information content of samples.
  3. Loadings per bicluster.
  4. Factors per bicluster.

    

  1. Several extract plots.
  2. Bicluster plot. This  plot is similar to heatmap plot. From the following options, one bicluster can be visualized by two plots:
        • Data matrix: The data matrix is sorted such that the bicluster appear at the upper left corner. The bicluster is
           marked by a rectangle.
        • Bicluster only: Only the given bicluster is plotted.
        • If both options are checked, the two plots will appear sequentially.
        • If the "opposite" option is checked, the opposite bicluster will be plotted.
