Data Loading

There are several ways to input the data set into the  package:
  • Enter data directly by choosing this option: Data > New data set.....
  • Load a data set from an R workspace : Data > Load data set....
  • Import data from a plain-text (ASCII) file or from another statistical package, such as Minitab, SPSS, or Stata:  Data > Import data
  • Read a dataset that is included in an R package: Data in packages

Users can load more than one dataset in the memory, however to perform the analysis, users need to specify a single active dataset.
