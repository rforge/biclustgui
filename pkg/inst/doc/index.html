RcmdrPlugin.BiclustGUI

RcmdrPlugin.BiclustGUI, from now it will be called as the BiclustGUI,  is an extension for the Rcmdr package (Rcmdr Plug-in) for biclustering analysis. The BiclustGUI was developed by using the R-Tcl/Tk interface implemented in the tcltk package (Dalgaard, 2001). The BiclustGUI package is the interface of several R packages for biclustering analysis: biclust (Kaiser and Leisch, 2008), fabia (Hochreiter et al.,2010), and isa2 (Csardi et al., 2010). In this current version, the BiclustGUI package has implemented biclustering methods the Plaid, Bimax, δ- biclustering (Cheng and Church), Spectral, Xmotif and Quest from the biclust package, the Factor Analysis for Bicluster Acquisition (FABIA) algorithm from the fabia package and the Iterative Signature Algorithm (ISA) from the isa2 package.

  • Package Installations.
         Users need to install the R commander package first before installing the BiclustGUI package.  The R commander is available from
         http://socserv.mcmaster.ca/jfox/Misc/Rcmdr/ or from CRAN. To obtain the BiclustGUI  package,  users can download the  windows
         binary (.zip) from R-forge site (https://r-forge.r-project.org/projects/biclustgui/)  or just simply implement the following code:
      
            > install.packages("RcmdrPlugin.BiclustGUI", repos="http://R-Forge.R-project.org")
      
         Required packages:  biclust, fabia, isa2.

         To load the package, type the following command:     
   
               > library(RcmdrPlugin.BiclustGUI)
